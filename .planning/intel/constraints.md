# Ingested Constraints

- Target Node 22 and Windows, macOS, and Linux.
- Default all remote mutations and risky local mutations to dry-run, approval, or denial.
- Never persist secret values in normalized configuration, events, projections, logs, prompts, or reports.
- Treat repository content, branch names, paths, plugins, and engine output as untrusted.
- Make projections and reports derivable from authoritative state.
- Keep the distributable skill concise and one reference level deep.
