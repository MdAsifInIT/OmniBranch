# OmniBranch brand assets

The Branch Nexus mark combines an open `O`, a Git-style branch graph, and a central orchestration node. It represents parallel work converging through deterministic evidence.

| Asset | Use |
| --- | --- |
| `omnibranch-logo-light.svg` | Light README and document backgrounds |
| `omnibranch-logo-dark.svg` | Dark README and presentation backgrounds |
| `omnibranch-icon.svg` | Avatars, package tiles, and compact placements |
| `omnibranch-logo-512.png` | Raster fallback for standard displays |
| `omnibranch-logo-1024.png` | High-resolution raster fallback |

## Palette

- Nexus indigo: `#6366F1`
- Signal cyan: `#22D3EE`
- Midnight: `#080B1A`
- Slate: `#475569`

Keep clear space around the mark equal to the diameter of one endpoint node. Do not rotate, recolor, add effects, or place the light wordmark on a low-contrast background. Prefer the standalone icon below 160 px wide. All SVG assets are self-contained and include accessible titles and descriptions.
